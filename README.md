# multiphase-flow-microfluidics
# Multiphase Flow Microfluidics

This project investigates water-in-gas droplet formation in microfluidic systems through computational fluid dynamics analysis and dimensionless number calculations.

The workflow was developed in Python to analyze the relationship between gas flow rate, liquid flow rate, and droplet generation behavior within microfluidic environments.

## Project Features
- Reynolds number calculations
- Weber number calculations
- Capillary number calculations
- Hydraulic diameter calculations
- Droplet volume and diameter estimation
- Flow regime characterization
- Experimental mapping analysis
- Regime transition analysis
- Gas and liquid flow rate conversions
- Data visualization and plotting

## Methods
Experimental data was processed using Python, NumPy, pandas, and matplotlib. The computational workflow converts flow parameters into SI units, calculates fluid dynamic properties, and generates plots to analyze droplet stability, production frequency, and multiphase flow behavior.

## Key Analysis Areas
- Droplet production trends
- Gas inertia effects on droplet volume
- Weber number sensitivity
- Experimental regime mapping
- Viscous versus inertial force balance
- Flow regime transition behavior
